# Sportomic Task

A small Next.js + TypeScript project for visualizing court selection and turf results.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the app in your browser:

```text
http://localhost:3000
```

## Build & Production

Build the project:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Notes

- This project uses Next.js with React and TypeScript.
